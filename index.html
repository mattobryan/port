MedLit Search

Live Demo
mattobryan.github.io/Medlit
6 databases · PICO search · AI synthesis

Clinical literature search across 6 free databases simultaneously, with one-click routed access to 8 paywalled databases and AI-powered evidence synthesis.

The problem: researchers and clinicians juggle six or more separate database search interfaces and manually re-run the same query on each one, then manually work out which results they can actually reach through their institution's paywall access. The approach: a single-file React app that queries PubMed, Europe PMC, Semantic Scholar, OpenAlex, CrossRef, and DOAJ in parallel, auto-detects study type, offers a structured PICO search mode, checks Unpaywall for legal open-access copies, and — with the visitor's own Anthropic API key — asks Claude to synthesise up to 20 papers into an evidence-quality summary. The outcome: a zero-install, zero-cost tool that ships as one static HTML file on GitHub Pages.

React 18 (CDN), Babel Standalone, PubMed / Europe PMC / Semantic Scholar / OpenAlex / CrossRef / DOAJ / Unpaywall APIs, Claude
